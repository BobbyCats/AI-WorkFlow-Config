# Claude Code 指令速查表

> 中文版斜杠命令大全，方便随时查阅。

## 核心指令

| 指令       | 用途          | 使用场景             |
| ---------- | ------------- | -------------------- |
| `/help`    | 显示帮助信息  | 不确定有哪些功能时   |
| `/config`  | 查看/修改配置 | 调整输出风格、主题等 |
| `/compact` | 压缩对话历史  | 对话过长时释放上下文 |
| `/clear`   | 清空当前对话  | 完全重新开始         |

## 模式切换

| 指令    | 用途          | 使用场景         |
| ------- | ------------- | ---------------- |
| `/plan` | 进入计划模式  | 复杂任务前先规划 |
| `/vim`  | 启用 Vim 键位 | Vim 用户提升效率 |

## 上下文管理

| 指令      | 用途             | 使用场景             |
| --------- | ---------------- | -------------------- |
| `/add`    | 添加文件到上下文 | 需要 AI 了解特定文件 |
| `/remove` | 从上下文移除文件 | 减少不必要的上下文   |
| `@文件名` | 引用文件         | 快速引用指定文件     |
| `@符号名` | 引用代码符号     | 引用函数、类等       |

## 输出控制

| 指令          | 用途         | 使用场景               |
| ------------- | ------------ | ---------------------- |
| `/statusline` | 自定义状态栏 | 显示上下文用量、分支等 |
| `/output`     | 切换输出风格 | 调整详细程度           |

## 高级功能

| 指令    | 用途          | 使用场景         |
| ------- | ------------- | ---------------- |
| `/mcp`  | 管理 MCP 连接 | 连接外部服务     |
| `/hook` | 管理 Hooks    | 自动化权限审批等 |

## 快捷键

| 按键      | 功能         |
| --------- | ------------ |
| `Tab`     | 自动补全     |
| `Ctrl+C`  | 取消当前操作 |
| `Ctrl+L`  | 清屏         |
| `↑` / `↓` | 历史记录导航 |

## 常用提示词模式

| 模式     | 示例               | 效果              |
| -------- | ------------------ | ----------------- |
| 计划优先 | "先制定计划再执行" | 提高一次成功率    |
| 挑战模式 | "严格审查这些变更" | 让 AI 当审查员    |
| 自主修复 | "修复失败的测试"   | AI 自动定位并修复 |
| 重构模式 | "实现更优雅的方案" | 让 AI 重新设计    |

## 提示词后缀

在任何请求后附加这些短语可以改变 AI 行为：

| 后缀             | 效果                 |
| ---------------- | -------------------- |
| "use subagents"  | 使用子智能体并行处理 |
| "先思考再执行"   | 让 AI 先分析再行动   |
| "更新 CLAUDE.md" | 让 AI 记录学到的教训 |
| "记录错误经验"   | 触发错误记录技能     |
