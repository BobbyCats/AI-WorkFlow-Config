# 重构提示词模板

## 优雅重构

```
基于你现在了解的一切，废弃当前方案，实现一个更优雅的解决方案。
要求：
- 保持功能不变
- 提高可读性和可维护性
- 减少重复代码
```

## 结构重组

```
请将这个文档/代码重新组织：
1. 按逻辑分组相关内容
2. 添加清晰的章节标题
3. 确保信息流向合理
4. 保留所有原有信息
```

## 精简优化

```
在保持完整性的前提下，精简以下内容：
- 删除冗余表达
- 合并重复信息
- 简化复杂句式
- 保留核心要点
```

## 模块化拆分

```
将这个大文件拆分为多个模块：
- 每个模块职责单一
- 模块间依赖清晰
- 提供统一的入口/索引
```
