# 代码审查提示词模板

## 通用审查

```
请审查以下代码/文档，关注：
1. 是否有逻辑错误或潜在问题
2. 是否符合项目规范
3. 是否有改进空间

审查完成后，按严重程度分类列出问题。
```

## 翻译质量审查

```
请对比原文和译文，检查：
1. 翻译准确性 - 是否忠实于原文
2. 表达自然度 - 是否符合中文表达习惯
3. 术语一致性 - 专有名词翻译是否统一
4. 格式完整性 - 链接、代码块是否正确

发现问题时，给出具体位置和修改建议。
```

## 严格审查模式

```
以 Staff Engineer 的标准严格审查这些变更：
- 指出所有潜在问题
- 不要放过任何细节
- 在我通过你的审查之前，不要提交
```
