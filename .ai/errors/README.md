# 错误记录示例

此目录用于存放开发过程中的错误经验记录。

## 记录格式

每个错误创建一个 Markdown 文件，命名格式：`YYYY-MM-DD-简短描述.md`

## 示例

```markdown
# 2024-01-15 replace_file_content 引号匹配失败

## 现象

使用 replace_file_content 工具时，目标内容找不到

## 原因

文件中的中文引号""和英文引号""编码不同，
即使看起来一样，实际上是不同的字符

## 解决方案

使用 PowerShell 按行号直接替换，避免字符匹配问题

## 教训

- 涉及引号时，优先使用按行号操作的方式
- 先用 grep 确认实际内容再替换
```
